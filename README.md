# Explore Instances - Sales Showcase

Minimal showcase version for sales demonstrations.

## Features

      - Browse all project instances
        - Search and filter by type, client, and features
          - Launch projects directly
            - Optimized for fast loading

## Development

    ```bash
npm install
npm run dev
```

The app runs on port 5124.

## Build

```bash
npm run build
```

The output will be in the `out` directory, ready for static hosting.

## Cursor Control Center

This project is configured to work with Cursor Control Center:
- Port: 5124
- Launch command: `npm run dev:web`
